# Archers

The following commands work only for Archers:

| Command | Description | Shorthand |
|---------|-------------|-----------|
| SIGHT \<DIR\> | Look for targets in this direction | SI \<D\> |

## Note

Archers are currently the only character class allowed to use the missile weapons (i.e. shortbows, longbows, and crossbows). No other character class is proficient in their use. Also see the `fire` command under the `combat` help topic.
